## Clone
```
git clone https://github.com/nethunter-indo/compiler

cd compiler

```

## Add Kernel Source
```
mkdir source && cd source
git clone <sourceurl> <devicecodename>

# Example :
git clone https://github.com/Hdr-ProjeKT/KT_a51 a51

cd ..
```

## Start

```./builder.sh```

## Getting update
